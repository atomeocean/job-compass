# Uber 前端 面试经验

<InterviewDetail />

## 基本信息

- **面试日期**：近期
- **面试岗位**：Software Engineer
- **学历背景**：硕士，偏后端，简历上有一年前端经历
- **工作类型**：全职
- **投递方式**：内推
- **面试形式**：Online Assessment + 技术电面
- **当前状态**：在职跳槽

## 面试详情

### 面试流程概览

- Round 1：技术电面 Coding
- Round 2：System Design
- Round 3：BQ
- Round 4：Coding 1
- Round 5：Coding 2（前端）

### 第一轮：技术电面
给定一个数组，元素均为 player 的 rank，胜负由 rank 决定的规则，输出锦标赛的结果。题目比较开放。最后问了一些关于数据库查询比赛记录的问题。

### 第二轮：System Design
设计 Google Calendar。

### 第三轮：BQ
非常规 BQ，问了很多实际工作细节，包括 UI 相关。面试官非常想找能直接上手工作的人。候选人偏后端，面试过程中状态不佳，未能正常发挥。

### 第四轮：Coding 1
LeetCode 380（Insert Delete GetRandom O(1)）。

### 第五轮：Coding 2（前端）
用 React 实现一个停车场系统。

## 面试结果反馈

- **当前状态**：被拒

## 面试体验

个人背景更偏后端，简历上有一年写前端经历也是无奈配合manager，面试过程中甚至触发了某种PTSD没有发挥好，没有回复，应该是默拒了。
能感觉BQ的manager和coding 2的组内面试官特别疲倦，但coding 1的面试官transfer过来的天竺人居然是在床上躺着面的lol。
