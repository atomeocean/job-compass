# Airbnb Software Engineer全流程面试经验与时间线

<InterviewDetail />

## 基本信息

- **岗位**：Software Engineer
- **面试形式**：Phone Screen + Virtual Onsite
- **申请渠道**：内推
- **候选人背景**：硕士，在职跳槽
- **面试结果**：Pass

## 时间线

### Day 1

Recruiter Call。

简单聊了一下背景经历，结束后直接安排 Phone Screen。

### Day 10

Phone Screen。

不是常见面经原题，后来发现对应 LeetCode 2101（Detonate the Maximum Bombs）。

#### Follow Up

如果炸弹（传播）可以沿对角线扩散，应该如何修改解法。

前面沟通花了比较多时间，导致 Follow Up 时间比较紧张。

最后虽然把思路和代码写出来了，但没来得及完整跑测试。

面试官人非常好，结束时还主动安慰不要担心。

第二天一早 Recruiter 就通知通过，并开始安排 VO Prep Call 和 VO。

---

### Day 15

VO Prep Call。

Recruiter 非常详细地分享了 Phone Screen 的反馈。

反馈内容：

- Overall 表现不错
- Follow Up 虽然没跑测试
- 但整个过程 Communication 非常好

Recruiter 也特别强调：

VO 包含：

- Coding
- Code Review
- System Design
- Project Deep Dive

以上四轮通过后，才会进入 Core Value 环节。

另外还特别提到：

- Coding 如果发挥不好，有机会申请加面
- System Design 如果挂掉，基本没有补救机会
- SD 的表现对于后续 HC 决策非常重要

---

### Day 29 & Day 30

Virtual Onsite。

#### Coding

经典 DP + Topological Sort 面经题。

有几个 Follow Up，但整体难度不大，过程比较顺畅。

#### Code Review

是地里常见的 Python Code Review 题。

代码量比较大，没有全部看完。

时间有限，只能尽量多写一些 Comment。

#### Project Deep Dive

围绕最有挑战性的项目展开。

除了项目本身的技术细节，还会不断穿插 BQ 问题。

特点：

- 挖得非常深
- 需要详细解释技术决策
- 需要说明团队协作过程
- 需要解释业务背景和影响

既考察技术，也考察沟通和影响力。

#### System Design

经典 Group Chat。

整体体验和很多面经描述一致：

Communication 非常重要。

面试官并不希望直接背模板答案。

更关注：

- Thinking Process
- Trade-off 分析
- 如何一步步推进设计

整个讨论过程比较顺畅。

不过时间还是有点紧张，最后没来得及展开讨论一些细节，例如：

- Database Index
- 更深入的数据建模方案

---

### Day 31

Recruiter 主动 Follow Up。

询问整体面试体验，并主动给出了预计反馈时间：

```text
2 ~ 3 天
```

第一次遇到 Recruiter 主动给 ETA，体验非常好。

---

### Day 34

Recruiter 通知 VO 通过。

开始安排 Core Value 面试。

同时提到已经开始准备 Team Match。

由于手里还有其他 Pending Offer，简单和 Recruiter 提了一下。

Recruiter 立刻开始加速流程。

---

### Day 36

Core Value。

和一位面试官进行了比较轻松的交流。

没有太多技术问题。

更像是：

- 价值观匹配
- 工作方式
- 团队协作风格

方面的深入聊天。

---

### Day 37

Team Match。

和多个组的 Hiring Manager 分别沟通。

整体体验非常好。

每个组都介绍了：

- 业务方向
- 团队现状
- 技术挑战

当天就完成 Team Match。

随后 Recruiter 开始准备送 HC。

---

### Day 39

HC（Hiring Committee）通过。

当天下午收到 Verbal Offer。

## 面试体验

整个流程最大的感受：

- Recruiter 非常专业且高效
- Communication 在 Airbnb 非常重要
- System Design 权重很高
- Project Deep Dive 会挖得非常深入
- 面试体验整体非常友好

虽然一直听说 Airbnb 流程慢，但这次体验非常顺畅。

从 Day 1 到 Verbal Offer 仅用了 39 天。

## 面试结果反馈

- **Phone Screen**：Pass
- **Virtual Onsite**：Pass
- **Core Value**：Pass
- **Team Match**：Pass
- **HC**：Approved
- **最终结果**：Offer
