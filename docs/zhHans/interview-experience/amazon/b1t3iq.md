# Amazon L5 Software Development Engineer Phone Screen面试经验

<InterviewDetail />

## 基本信息

- **岗位**：Software Development Engineer（L5）
- **面试形式**：Phone Screen
- **申请渠道**：内推
- **候选人背景**：硕士，在职跳槽
- **面试结果**：Fail

## 面试详情

整场面试包含三部分：

- Behavioral Questions（BQ）
- Coding
- System Design

### Behavioral Questions

先进行了常规的 Behavioral Questions，围绕项目经历和工作经验展开。

### Coding

算法题为 LeetCode 98（Validate Binary Search Tree）。

现场写出了时间复杂度和空间复杂度都符合要求的解法，不过实现方式相对复杂一些。

个人感觉可能存在以下几种情况：

- 解法解释得不够清楚；
- 面试官没有完全理解实现思路；
- 花费时间较多，导致没有机会进入第二道 Coding 题。

### System Design

第一轮还包含了一道小型 System Design。

设计题与组内实际业务相关，围绕团队当前负责的系统展开。

比较意外的是，Recruiter 在面试前并没有提到第一轮会涉及 System Design，因此没有针对性准备。

## 面试体验

整体难度中等。

除了算法之外，还需要准备：

- Behavioral Questions
- 与团队业务相关的小型 System Design

如果是组招（Team Matching）岗位，建议提前准备一些基础的系统设计内容，即使 Recruiter 没有提前说明。

## 面试结果反馈

- **最终结果**：Fail
