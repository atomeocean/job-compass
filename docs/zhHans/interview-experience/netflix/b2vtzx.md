# HR screening chat面试经验

<InterviewDetail />

## 基本信息

- **岗位**：Software Engineer
- **面试形式**：HR screening chat（30分钟）
- **申请渠道**：LinkedIn recruiter主动联系
- **候选人背景**：硕士，在职跳槽
- **面试结果**：Pass（进入后续60分钟coding电面）

## 面试内容

这轮是 recruiter 主动在 LinkedIn reach out，然后约了一个 30 分钟的 HR chat，主要是 general hire / ads 组方向。

### 常规问题

- 为什么要换工作？
- 介绍一下过去的工作经历，以及具体做过的项目
- 工作中遇到的最大挑战是什么
- 从 culture memo 里选一个点展开（选择的是 selflessness）
- 对 candor 的理解，以及最近收到过的反馈或建议

### 补充问题

- 是否需要 visa sponsorship
- 是否接受 hybrid / remote
- 期望薪资范围
- 上一份工作使用的技术栈 / 编程语言

### 面试体验

Netflix culture memo 问得比较深入，尤其是 candor 和 selflessness 会追问细节，不只是简单讲概念。

整体更像结构化聊天，不是技术面，但节奏比较紧凑。

最后 HR 表示会进入下一步，安排 60 分钟 coding 电面。

## 面试感受

7 年没有面试，这一轮明显有点紧张，表达有点磕磕巴巴，但整体沟通没有问题。

## 面试结果反馈

- **最终结果**：Pass
- **后续流程**：进入 coding phone screen
