# ByteDance SRE 面试经验

<InterviewDetail />

## 1. 基本信息

- 公司信息：ByteDance
- 面试时间：2026年5月
- 候选人背景：码农类General，硕士，在职跳槽
- 投递渠道：猎头 / HR 在 LinkedIn 上主动联系
- 结果：Fail（第二轮后收到 Thank You letter）

## 面试详情

原计划四轮，每轮一小时，第四轮为 Hiring Manager 面。先约前两轮，过了再约三四轮。实际只进行了两轮即被拒。

- Round 1：项目深挖 + System Design + Coding
- Round 2：Linux 理论 + Shell Script + Coding

### 第一轮：项目 + System Design + Coding

先聊简历上的项目和整体经历，与 JD 比较 match，聊得比较顺畅，约半小时。然后一道比较简单的 system design，与候选人 project 相关（应为巧合）。最后剩约十分钟做一道 LeetCode 简单题。

字节 VO 写题完成后需要直接跑 test case，15 个 test case 一遍通过。

### 第二轮：Linux 理论 + Shell Script + Coding

简单自我介绍后直接考 Linux 理论知识，约半小时，问得比较深入。大部分是高频问题，但稍微深入或拓展就比较吃力。面试官引导耐心、会讲解。

随后要求现场写一个 shell script，未能写出。最后剩余时间做了一道 LeetCode 中等难度变种题。

## 面试结果反馈

- **最终结果**：第二天收到 Thank You letter。
