# Google R1 Software Engineer电话面试经验

<InterviewDetail />

## 基本信息

- **面试日期**：近期
- **岗位**：Software Engineer（R1）
- **面试形式**：技术电面
- **申请渠道**：网上海投
- **候选人背景**：硕士，在职跳槽
- **面试结果**：Fail

## 面试详情

### Coding

题目类似 LeetCode 236（Lowest Common Ancestor）的变体。

给定马匹之间的族谱关系，以及两匹马的信息，判断这两匹马是否存在血缘关系。

输入：
- 马的族谱关系
- 两匹马

输出：
- 是否存在血缘关系

族谱关系的数据结构可以自行设计。面试中使用的是 List 结构，也可以使用 Tree 结构进行建模。

### Behavioral Questions

这轮 BQ 和网上常见的 Google BQ 不太一样。

问题数量不算多，但每个问题都会不断深挖，可能也和面试官个人风格有关。

#### 问题一

如果你在一家团队氛围和 Work-Life Balance 都比较差的公司工作，你会怎么做来改变这种情况？

Follow Up：

如果你是这家公司的管理者，你会怎么做？

#### 问题二

描述一次主动承担职责范围之外工作的经历。

#### 问题三

描述一次使用更有创造性的方式解决问题的经历。

## 面试体验

Coding 部分难度不高，属于树和祖先关系相关问题的变体。

BQ 部分相比预期占比更高，虽然问题数量不多，但每个问题都会持续深挖细节，需要准备比较完整的案例和思考过程。

## 面试结果反馈

- **最终结果**：Fail
