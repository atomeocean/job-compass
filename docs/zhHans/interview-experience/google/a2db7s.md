# Google公司L4 AI/ML Engineer Onsite面试经验

<InterviewDetail />

## 基本信息

- **面试日期**：近期
- **面试岗位**：AI/ML Engineer
- **岗位级别**：L4
- **学历背景**：硕士
- **工作类型**：全职
- **投递方式**：内推
- **面试形式**：Onsite
- **当前状态**：在职跳槽

## 面试详情

### 第一轮 BQ

正常 BQ，需要准备一下 conflict，如何 mentor 不太行的组员，都比较常见。

### 第二轮 AI/ML domain specific

先问了下 project dive，（lz 现在在做 agent）follow up：agent 表现不好的时候怎么分析。

ML design：设计一个 mobile app，拍照然后 detect leaf category，retrieval + ranking

### 第三轮 coding

LC 1101，找朋友，follow up：如果也有 unfriend，怎么做？建图 + BFS。

### 第四轮 coding

necklace，这题比较少见，一个由 D 和 R 组成的字符串，#D = #R，cut 之后分给两个人，保证两个人的 D 和 R 相等

关键在于分析出来需要 at most 2 cuts + sliding window。

## 面试结果反馈

一周后 HR feedback positive，但有可能 downlevel 到 L3，进入 team match。

因为后来 match 了 SDE track，感觉也 open to SDE，决定加面一轮 coding 再送 HC。
