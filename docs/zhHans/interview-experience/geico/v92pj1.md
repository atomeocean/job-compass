## 面试详情

### 第一轮 coding

矩阵乘积 类似 lc 311

### 第二轮 AI Coding

这一轮并不是Meta 那样的AI Assisted Coding, 而是类似AI Design。题目要求给一堆doc，实现一个基础的RAG 框架。当时讨论了具体embedding, Elastic Search index, similarity search, prompting, evaluation 等等。

### 第三轮 System Design

设计视频网站，需要支持搜索，根据用户兴趣进行推荐。面试官显然对”搜索“和”推荐“更感兴趣，对视频那一块没有深究。

### 第四轮 BQ

老生常谈，conflict, constructive feedback等
