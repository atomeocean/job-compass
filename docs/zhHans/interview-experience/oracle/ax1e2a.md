# Oracle健康组技术电面面经

<InterviewDetail />

## 基本信息

- **岗位**：Software Engineer
- **面试形式**：技术电面
- **申请渠道**：网上海投
- **候选人背景**：硕士，在职跳槽
- **面试结果**：Fail

## 面试流程

### 自我介绍

简单自我介绍后进入技术和行为面试环节。

### Behavioral Questions

- 如何处理团队冲突（conflict）
- 如何与他人合作解决复杂问题

### Design Question

- 大文件上传如何提升容错能力
- find missing number（缺失数字问题）

设计题偏基础系统思考 + 算法混合类型。

### Coding

题目是 LeetCode 简化版本 735，整体难度不算高，但需要注意细节实现。

## 面试结果反馈

面试结束第二天早上收到拒信。

## 面试总结

整体难度偏高，节奏比较紧张。

最大的教训是：`LeetCode 还是要保持熟练度。`
