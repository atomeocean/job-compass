# 美国留子的失业罪

后退一步看，牢美始终在用不同形式给留子定失业罪的罪名。

现在正在用看似不尖锐的形式包装。“你可以失业，这里有包容的牌坊。”

opt允许你失业90天，H1b允许你失业60天。

名义不同，但本质上，这些都是失业罪。
只为了从将中国留学生尽快驱逐出去。

只要你失业了，立刻套上罪名指控。

那么自然想到了，中国留学生在美国正在面临的一项困境，毕业罪。
