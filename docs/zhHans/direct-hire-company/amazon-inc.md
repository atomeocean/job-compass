# 亚马逊 Amazon Inc.

总部位于美国西雅图。

## 工作环境
亚马逊是一个规模非常庞大的公司，其中AWS部门是它的核心部门。

中国留学生普遍在亚马逊的技术相关部门工作，因此不讨论亚马逊的仓库管理、物流等非技术部门，尽管这些部门以rapper多，工作时长过长著称。
类似的，亚马逊的技术部门也是血汗工厂，工作强度大，加班多，压力大。部门负责的业务越成熟，接的需求就越符合拧螺丝钉，不仅耗费精力，还没有成就感。这里面最恶心的部门当属AWS EC2部门，不要去。
