# 互联网公司怎么拿到好的绩效

这个话题有些敏感，但是应该人人关心。我工作10年了，从没有拿到过“平凡”绩效，这件事情我有一点点发言权。这并不容易，一般刚毕业的同学不容易拿到好的绩效，他需要时间去证明自己和建立信任；跳槽后呢，也是类似的，熟悉陌生的环境和与他人建立信任都需要时间；除此之外，一般晋升之后，也会有一次“平凡”绩效反馈。我不太聪明，也没啥大的成就，所以要么是运气好，要么是还是有值得借鉴的方法。
