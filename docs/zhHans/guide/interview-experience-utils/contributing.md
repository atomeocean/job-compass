# 面试经验贡献指南

## 阅读须知

1. 本文是一份**简要指南**，只概述基本流程和规则。完整的操作步骤请参考 [Atomeocean面经贡献流程指南](https://logbook.atomeocean.com/guide/light-weight-work/ceremonial-contribution/interview-experience-contribution.html)。
2. **设备要求**：请使用笔记本电脑或台式电脑完成贡献流程，不支持 iPad、iPhone 等移动设备。
3. **账号要求**：需要注册并登录 [GitHub](https://github.com/) 账号。

欢迎为Job Compass贡献面试经验！您的分享将帮助更多求职者更好地准备面试。请遵循以下规则和流程。

## 基本步骤

1. **准备内容**：整理面试基本信息（JSON 文件）和面试详情（Markdown 文章），并标注引用来源。
2. **提交 PR**：通过 GitHub 向 Job Compass 项目提交 Pull Request，每个 PR 仅包含一篇面试经验。
3. **等待审核**：PR 需通过自动编译检查，并由至少一位社区成员人工审核确认。
4. **合并发布**：审核通过后，PR 将被合并至 main 分支，部署成功后文章自动发布至官网面经页面。

## 贡献原则

1. **真实性**：请确保分享的面试经历是您亲身经历或经确认真实的。
2. **隐私保护**：
   - 请勿泄露作者的真实姓名、联系方式或其他个人非公开信息。
3. **客观友好**：描述面试过程时请保持客观，避免使用攻击性的语言。
4. **拒绝灌水**：AI工具仅限于协助生成Json文件。
