# 面试经验贡献指南

完整流程请参考 [Atomeocean面经贡献流程指南](https://logbook.atomeocean.com/guide/light-weight-work/ceremonial-contribution/interview-experience-contribution.html)。

欢迎为Job Compass贡献面试经验！您的分享将帮助更多求职者更好地准备面试。请遵循以下规则和流程。

## 贡献原则

1. **真实性**：请确保分享的面试经历是您亲身经历或经确认真实的。
2. **隐私保护**：
   - 请勿泄露作者的真实姓名、联系方式或其他个人非公开信息。
3. **客观友好**：描述面试过程时请保持客观，避免使用攻击性的语言。
4. **格式规范**：请按照推荐的模板格式撰写，以便于阅读和归档。
