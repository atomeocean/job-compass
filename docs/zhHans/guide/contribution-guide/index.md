# Job Compass 贡献指南

欢迎参与 Job Compass 开源项目！本手册汇总了向项目贡献内容所需的操作步骤、实用技巧和常见问题排查方法。

## 快速开始

| 步骤 | 操作 | 说明 |
|:---:|---|---|
| 1 | [Fork 并克隆仓库](/guide/github-setup-guide.md) | 完成本地 Git 环境配置 |
| 2 | 新建分支，进行内容修改 | 参考[贡献技巧](/guide/contribution-guide/tips.md)了解命名规范 |
| 3 | [提交 Pull Request](/guide/github-pull-request-guide.md) | 填写标题和描述，指定 Reviewer |

## 本手册包含

| 文档 | 内容 |
|---|---|

| [常见问题排查](/guide/contribution-guide/troubleshooting.md) | 侧边栏不显示、构建报错、组件无数据等问题的解决方案 |
